# 闪拍证件照 (FlashPhoto)

AI智能证件照 + 艺术写真 微信小程序

## 做什么

一款基于AI的证件照制作小程序。用户上传自拍照，AI自动检测人脸、更换背景、调整尺寸，3秒生成标准证件照。同时提供AI艺术写真功能。

## 为什么做

- 传统照相馆证件照 20-50元，AI证件照仅需 1-3元，价格优势巨大
- 证件照是刚需（考试报名、签证申请、入职体检、社保办理）
- 微信2026年AI小程序成长计划提供免费云开发资源和AI算力
- AI图片处理是已验证的赚钱赛道

## 目标用户

- 需要证件照的普通用户（学生、求职者、出国人群）
- 喜欢AI写真的年轻女性用户
- 追求性价比的手机用户

## 预期收益

- 首月目标：1000+用户，1000-5000元收入
- 变现模式：按次付费 + VIP会员 + 激励广告

## 技术栈

- 微信原生小程序
- 微信云开发（云函数 + 云数据库 + 云存储）
- 腾讯混元AI / 腾讯云AI图像处理
