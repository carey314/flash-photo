# 闪拍证件照 — 部署上线指南

## 前置准备（Ying 操作）

### 1. 微信小程序账号

**已有小程序账号：**
直接跳到第2步。

**没有账号：**
1. 打开 https://mp.weixin.qq.com/
2. 点击「立即注册」→ 选择「小程序」
3. 推荐使用企业主体（个体工商户也可以），原因：
   - 个人主体**不能开通微信支付**（核心变现渠道）
   - 个人主体**不能开通流量主**（广告收入）
   - 注册个体工商户成本低（部分地区免费），可在线办理
4. 注册完成后，记下 **AppID**

### 2. 申请 AI 小程序成长计划

1. 登录小程序后台 https://mp.weixin.qq.com/
2. 左侧菜单：「行业能力」→「AI小程序成长计划」
3. 点击报名参与
4. 获得的免费资源：
   - 6个月云开发环境
   - 1亿混元Token
   - 1万张文生图额度
   - We分析专业版一年

**截止日期：2026-06-30，尽早申请！**

### 3. 开通微信云开发

1. 下载微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 打开开发者工具 → 导入项目
   - 项目路径：`/Users/carey/projects/projects/flash-photo`
   - AppID：填入你的小程序 AppID
3. 点击顶部「云开发」按钮
4. 开通云开发环境（选择成长计划的免费环境）
5. 记下**环境ID**

### 4. 配置 AppID

编辑 `project.config.json`，填入 AppID：
```json
{
  "appid": "你的AppID",
  ...
}
```

## 部署步骤

### Step 1: 初始化数据库

1. 在开发者工具中，右键点击 `cloudfunctions/initDB`
2. 选择「上传并部署：云端安装依赖」
3. 部署完成后，右键 → 「在终端中调用」
4. 确认返回 4 个集合创建成功

### Step 2: 部署所有云函数

依次右键部署以下云函数（选择「上传并部署：云端安装依赖」）：

1. `cloudfunctions/processPhoto` — 照片处理（核心）
2. `cloudfunctions/createOrder` — 创建订单
3. `cloudfunctions/payCallback` — 支付回调
4. `cloudfunctions/generateLayout` — 排版生成
5. `cloudfunctions/getUserInfo` — 用户信息

### Step 3: 配置腾讯云 AI（可选，增强抠图效果）

1. 登录腾讯云控制台 https://console.cloud.tencent.com/
2. 开通「人体分析」服务（有免费额度）
3. 获取 SecretId 和 SecretKey
4. 在云开发控制台 → 云函数 → `processPhoto` → 编辑
5. 添加环境变量：
   - `TENCENT_SECRET_ID` = 你的SecretId
   - `TENCENT_SECRET_KEY` = 你的SecretKey

**不配置也能用**：系统会使用基础背景替换方案。

### Step 4: 配置微信支付（企业主体）

1. 申请微信商户号：https://pay.weixin.qq.com/
2. 小程序后台 → 微信支付 → 关联商户号
3. 编辑 `cloudfunctions/createOrder/index.js`：
   - 填入 `subMchId`（商户号）
4. 重新部署 `createOrder` 云函数

**暂时不配支付也能测试**：免费体验功能正常，付费功能暂时不可用。

### Step 5: 本地调试

1. 在开发者工具中点击「编译」
2. 测试流程：
   - 首页 → 拍照/选照片 → 选规格选背景 → 生成 → 预览
3. 检查控制台是否有报错
4. 确认云函数调用正常

### Step 6: 提交审核

**审核前检查清单：**

- [ ] 所有页面正常显示，无空白页
- [ ] 拍照和相册选择功能正常
- [ ] 证件照生成流程跑通
- [ ] 隐私协议已添加（涉及相机、相册权限）
- [ ] 用户协议已添加
- [ ] tabBar图标正常显示

**AI类目资质（重要）：**

如果使用AI功能，需要在小程序后台添加类目：
- 类目：IT科技 → 数据处理
- 或：生活服务 → 摄影/照片冲印

如果涉及深度合成（AI生成/修改人脸）：
- 需要补充「深度合成类目」
- 可能需要算法备案（使用腾讯云API的话，腾讯已备案，通常不需要开发者单独备案）

**提交审核：**
1. 开发者工具 → 上传代码
2. 小程序后台 → 版本管理 → 提交审核
3. 审核周期：通常1-3个工作日

## 上线后运营

### 流量主（广告收入）

小程序累计独立访客达到1000后可申请流量主：
1. 小程序后台 → 流量主 → 开通
2. 获取广告位ID
3. 更新 `result.js` 中的 `adUnitId`

### 数据监控

- 云开发控制台：查看云函数调用次数、数据库使用量
- We分析：查看用户数据、留存、转化
- 收入监控：查看 `revenue` 集合统计
